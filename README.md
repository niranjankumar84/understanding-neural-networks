# Understanding Neural Networks Using PyTorch

This repository contains my beginner Deep Learning experiments and implementations using PyTorch.

Currently, this project focuses on understanding how a single artificial neuron works internally using tensors, weights, bias, and forward propagation.

---

# Project Overview

In this notebook, I implemented a simple neuron using PyTorch and explored how neural networks perform mathematical computations behind the scenes.

The project helped me understand:
- Tensor operations
- Artificial neurons
- Weights and bias
- Forward propagation
- nn.Linear layer
- Neural network computation flow
